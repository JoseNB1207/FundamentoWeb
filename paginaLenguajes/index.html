JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Se usa para manipular el DOM, realizar peticiones a servidores y desarrollar aplicaciones web dinámicas.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilos que controla la presentación de las páginas web. Permite definir colores, fuentes, tamaños, posiciones y más para mejorar la apariencia del contenido HTML.

Atributos CSS

Nombre Descripción Sintaxis Imagen Fuente
color Define el color del texto. color: red; MDN
background-color Establece el color de fondo. background-color: blue; MDN
font-size Controla el tamaño del texto. font-size: 16px; MDN
margin Define los márgenes alrededor de un elemento. margin: 10px; MDN
padding Controla el espacio interno de un elemento. padding: 10px; MDN
border Define un borde alrededor de un elemento. border: 2px solid black; MDN
text-align Alinea el texto dentro de un elemento. text-align: center; MDN
display Define cómo se muestra un elemento. display: flex; MDN
position Controla la posición de un elemento en la página. position: absolute; MDN
box-shadow Agrega sombras alrededor de un elemento. box-shadow: 5px 5px 10px gray; MDN
